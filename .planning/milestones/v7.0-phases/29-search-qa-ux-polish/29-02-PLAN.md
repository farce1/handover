  **OSC8 helper function:**
  1. Add a helper function at the top of `search.ts`:
     ```typescript
     function formatSourceLink(sourceFile: string, isTty: boolean, outputDir: string): string {
       if (!isTty) {
         return sourceFile;
       }
       const absolutePath = path.resolve(outputDir, sourceFile);
       const fileUrl = `file://${absolutePath}`;
       return `\x1B]8;;${fileUrl}\x1B\\${sourceFile}\x1B]8;;\x1B\\`;
     }
     ```
  2. Import `path` from `node:path` at the top.
  3. Import `loadConfig` already exists — extract `config.output` (outputDir) to resolve paths. The `outputDir` should be derived from the config that's already loaded in `runSearch`. Pass it through to `runFastMode` and `runQaMode`.
